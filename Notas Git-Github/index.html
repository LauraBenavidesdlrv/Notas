NOTAS DE GIT / GITHUB

¿Qué es Git?

Es el sistema de control de versiones más utilizado, permite tener un historial completo del código que desarrollamos para una aplicación.

Git es un Sistema de Control de Versiones Distribuido (DVCS) o descentralizado, esto quiere decir que si en algún momento el servidor tiene un fallo; los programadores no solo podrán tener una copia instantanea de los archivos, sino que se replica el repositorio en su totalidad y permitiendo que sigan trabajando en sus códigos, de tal forma que cuando el servidor se recupere sea posible utilizar los repositorios y avances de los programadores para restaurarlo.


Usos de git

  • Historial de todo el trabajo que una o más personas haan realizado para una aplicación en cuestión
  • Almacenar código
  • Trabajar en equipo de manera fácil, remota y rápida
  • Permite saber en que parte del proyecto hay un error (el commit específico en el que está el fallo).

NOTA: Existen programas y aplicaciones para trabajar con git de manera gráfica, pero lo más recomendable es hacerlo desde la consola ya que dichos programas pueden estar un poco limitados en ciertas funcionalidades

Instalación

A continuación te muestro como instalar git

Pasos de instalación:

  1. Ir a la web oficial de Git https://git-scm.com/
  2. Buacar la sección de descarga.

  3. Según tu tipo de sistema operativo y preferencias descarga el instalador.
  4. Si eres usuario de Mac o Linux abres el instalador, le das en continuar a todo y por último install.
  5. Si eres usuario de Windows tienes diferentes opciones de configuracion antes de instalar. Aquí tienes un video tutorial para hacer la instalación en windows (ir al minuto 1:23).
  6. Una vez terminada la instalación:
    • Los usuarios de Windows pueden presionar clic derecho en la carpeta en donde deseen almacenar el proyecto y abrir el Git Bash.
    • Los usuarios de Mac pueden presionar la tecla cmd + epacio y escribir teminal.

  7. Una vez abierta la consola puedes proceder a configurar tu git.
